About this site

あそびをつくる人と、
関わる人をつなぐ。

ASOBI CREATE PARKは、地域のあそびやイベントを企画する人と、
参加したい人・手伝いたい人をつなぐマッチングサイトです。

🌿 ASOBI CREATE PARKとは

学校・PTA・おやじの会・町会・自治会・商店街・地域団体・個人企画者など、地域で「何かやってみたい」と思う人たちが、企画のヒントを見つけたり、仲間を集めたり、イベントを広げたりできる場所を目指しています。

💬 はじまりのきっかけ
「他の地域では何をやっているの？」
という声から。

地域には、たくさんの面白いあそびやイベントがあります。水あそび、外あそび、防災体験、工作、まつり、屋台、自然体験、親子イベント。けれど、その多くは、実施した団体の中だけで終わってしまうことがあります。

「あのイベント、どうやって準備したんだろう」
「費用はどれくらいかかったんだろう」
「何人くらいで運営したんだろう」
「うちの地域でもできるだろうか」
「手伝ってくれる人がいたら、できるかもしれない」

そんな声をつなげるために、ASOBI CREATE PARKは生まれました。

✨ ひとつのイベントが次のヒントに
地域のあそびは、
誰かの「やってみたい」から始まります。

でも、最初の一歩は簡単ではありません。場所の調整、道具の準備、人手の確保、安全管理、告知、当日の運営。ひとつのイベントの裏側には、たくさんの工夫と経験があります。

ASOBI CREATE PARKでは、そうしたイベントの事例やノウハウを共有し、これから企画したい人が参考にできるようにします。誰かが実施したあそびが、次の地域の「やってみたい」につながる。それが、このサイトの大切な役割です。

🎯 できること
01
あそびの事例・ヒントを見る
過去に実施された地域イベントの事例を見て、企画の参考にできます。対象年齢、準備物、人数体制、費用感なども確認できます。
02
あそびを企画・登録する
地域で実施したいイベントを登録し、参加者・スタッフ・協力団体を募集できます。
03
あそびに参加する
地域で開催されるイベントを探して、親子や友人同士で参加できます。地域とつながるきっかけが生まれます。
04
スタッフとして手伝う
「受付だけ」「設営・片付けなら」「写真撮影なら」—小さな関わり方も、地域イベントにとっては大きな力になります。
👥 対象となる人・団体
地域のあそびをつくりたい人、広げたい人、支えたい人に開かれた場所です。
学校・PTA関係者 おやじの会 町会・自治会 商店街 子育て団体 地域活動団体 NPO 個人企画者 企業・行政と連携した地域活動 地域イベントを手伝いたい人 親子で参加したい人

大切なのは、団体の大きさではありません。「地域で何かをやってみたい」「子どもたちや地域の人に楽しんでもらいたい」「自分たちの経験を、次の誰かの役に立てたい」—そんな思いがあることです。

📋 登録・利用について
⚠️ 登録情報は公開前提です

登録された団体情報・イベント情報・問い合わせ先は、原則としてサイト上に公開されます。個人用メールではなく、団体用・活動用・問い合わせ専用のメールアドレスの利用をおすすめします。

ASOBI CREATE PARKでは、安心して利用できる場所にするため、団体登録時に確認を行います。登録された問い合わせ先に運営から連絡し、正しく確認が取れた団体・活動のみ公開します。掲載内容がサイトの目的に合わない場合や、虚偽情報・なりすまし・危険な内容・過度な営業や勧誘などが確認された場合は、掲載を見送ることがあります。

🌿
地域のあそびの経験が蓄積され、
企画したい人がヒントを得られ、
参加したい人が出会い、
手伝いたい人が関われる。

ひとつの学校、ひとつの団体、ひとつの地域だけではできないことも、誰かの経験や手助けがあれば実現できるかもしれません。あそびは、まちをつなぐ力になります。ASOBI CREATE PARKは、そのつながりを少しずつ広げていきます。

みんなの「やってみたい」が、未来のあそび場をつくります。

ASOBI CREATE PARK 掲載団体利用規約（抜粋）

第1条（目的）
本規約は、ASOBI CREATE PARK（以下「本プラットフォーム」）に団体として掲載・活動するにあたって守るべきルールを定めます。

第2条（掲載の基準）
掲載できる団体は、世田谷区内またはその周辺で地域に根ざした活動を行っている団体とします。営利目的のみを目的とする団体は対象外となる場合があります。

第3条（情報の正確性）
登録する団体情報・イベント情報は正確な内容を入力してください。虚偽の情報が確認された場合は掲載を取り消す場合があります。

第4条（コミュニティへの貢献）
掲載団体は、地域の子どもたちや住民が安心して参加できる環境づくりに協力してください。

第5条（個人情報の取扱い）
参加申込者から受け取った個人情報は、イベント運営の目的のみに使用し、第三者への提供はしないものとします。

第6条（審査・掲載取消）
事務局は申請内容を確認し、掲載の可否を判断する権限を持ちます。また掲載後であっても規約違反が確認された場合は掲載を取り消すことができます。
📋 審査について
申請後、通常3〜7営業日以内にご登録のメールアドレスへ審査結果をお送りします。承認後は自由にイベントを掲載できます。
